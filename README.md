## Descripción
Este trabajo práctico fue desarrollado para la materia Paradigmas y Lenguajes.
Implementamos un sistema de semáforos en Common Lisp aplicando programación
funcional: funciones puras, sin variables globales y sin bucles imperativos.
También integramos la librería local-time via Quicklisp y reimplementamos
partes del sistema en Erlang para comparar ambos lenguajes.
