## Descripción
Este trabajo práctico fue desarrollado por estudiantes de la Universidad Nacional del Nordeste para la materia Paradigmas y Lenguajes, en la que se nos pidió implementar un sistema de semáforos en Common Lisp, aplicando programación
funcional: funciones puras, sin variables globales y sin bucles imperativos.
Además integrar la librería local-time via Quicklisp y reimplementar
partes del sistema en Erlang, nuestro lenguaje asignado, para comparar ambos lenguajes.

## Estructura del proyecto
